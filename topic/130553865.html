标题：开年帖！盘点一下从2010年左右至今的华人动画短片作品（持续更新）

各位新年快乐。楼主去年也发过类似的帖子，无奈被吞，今年再发一次，内容更多更精彩。

本来是写国产动画短片，后来发现也把华裔作品收录进去，于是就写华人作品啦。

为什么选择2010年左右的作品，主要从那时候接触动画短片吧。

ps.作品尽量以时间和工作室分类，都是楼主看过的作品。

8.4
2014 / 中国大陆 / 动画 奇幻 短片 / 耿雪

陶瓷动画《海公子》（这部应该也算定格吧）视频能搜得到，但画质感人。成人向的风格，恐怖类，啪戏√，cult√ 动画来源于《聊斋》的故事，原文： 东海古迹岛，有五色耐冬花，四时不调。而岛中古无居人，人亦罕到之。 登州张生，好奇，喜游猎。闻其佳胜，备酒食，自掉扁舟而往。至则 花正繁，香闻数里；树有大至十余围者。反复留连，甚慊所好。开尊 自酌，恨无同游。忽花中一丽人来，红裳眩目，略无伦比。见张，笑曰：“妾自谓兴致不凡，不图先有同调。”张惊问：“何人？”曰：“我胶娼也。适从海公子来。彼寻胜翱翔，妾以艰于步履，故留此耳。”张方苦寂，得美人，大悦，招坐共饮。女言词温婉，荡人神志。张爱好之。 恐海公子来，不得尽欢，因挽与乱。女忻从之。相狎未已，忽闻风肃肃，草 木偃折有声。女急推张起，曰：“海公子至矣。”张束衣愕顾，女已失 去。旋见一大蛇，自丛树中出，粗于巨筒。张惧，幛身大树后，冀蛇不 睹。蛇近前，以身绕人并树，纠缠数匝；两臂直束胯间，不可少屈。 昂其首，以舌刺张鼻。鼻血下注，流地上成洼，乃俯就饮之。张自分必死， 忽忆腰中佩荷囊，有毒狐药，因以二指夹出，破裹堆掌中；又侧颈自顾其掌， 令血滴药上，顷刻盈把。蛇果就掌吸饮。饮未及尽，遽伸其体，摆尾若霹雳 声，触树，树半体崩落，蛇卧地如梁而毙矣。张亦眩莫能起，移时方苏。 载蛇而归。大病月余。疑女子亦蛇精也。

莲花灯 (2017)
暂无评分
2017 / 中国大陆 / 动画 短片 / 沈星培

试验性特别强的作品

导演沈星培的这部动画作品《莲花灯》是他写给周璇的一封信，在十余个国际电影节公开展映，外媒评价其极具中国传统特色，又巧妙的介入了敏感的主题。 上海三四十年代诞生出的一大批歌后，诸如周璇、白光、姚莉、李香兰、葛兰等等，在进入二十一世纪之后，被年轻一代的性少数者，后被“追封”为 g/a/y icon 。 短片中，主角以莲花灯作为媒介，进入了镜子中的世界。镜子不但成为一个实行“法术”的法器，也是自恋行为的重要道具，完美地链接了真实与幻想、爱慕与自恋。 这种颇具恐怖片元素的手法不但在美学上暗合了导演的视觉风格，也因其包含的“悼念”、“故人”等元素，和年代久远的留声机音效的衬托下，更添诡异气氛。（出自某次看到的鹅厂影评，网址忘了）

相思 (2016)
8.2
2016 / 中国大陆 / 动画 短片 / 彭擎政 / 何向东 沈念慈

这部短片当时讨论度挺大的，但还没完结。楼主看完之后找过相关资料，来个剧透猜测。 给一些有关《相思》的幕后花絮，摘自豆瓣影评：王初桐的六娘，多少人的相思（相思）影评 　　https://movie.douban.com/review/8285350/　　版权归作者所有，任何形式转载请联系作者。 　　作者：人闲桂花落（来自豆瓣） 　　来源：https://movie.douban.com/review/8285350/ 　　动画《相思》的故事背景取材于清嘉定宝山文人毛大瀛（1735—1800）所著《戏鸥居词话》之《王竹所寄怀六娘词》篇。原文如下： 　　嘉定有六娘者，名湘苹，字采于，名家女也，归某生。某生狂荡无检，家产奁资，挥霍净尽，给六娘至勾栏迫之。六娘涕泗交横，号恸欲绝，假母以计污焉，遂堕入娼家。 　　后居槎水上，卖珠补屋，种竹浇花，幽窗曲几之下，熏炉茗碗间，静若书生。悦吾友王君竹所之才思，委身而不得，作小影以贻之。图中用“天寒翠袖薄，日暮倚修竹”两句补景。竹所署以“绝代佳人”四字，携入都门。一时知名之士，题咏殆偏。凡睹崔徽风貌者，皆以为远过萧娘一纸书也。 　　竹所有别六娘词，调寄《白苹香》云：“歌罢云分雨散，酒醒月黑风多。销魂无奈别离何。不是不曾真个。 宿粉未销衣袂，馀香犹在巾罗。橹声咿轧满烟波。一夜拥衾愁坐。”…… 然后楼主给那段古文大致翻译一下： 　　六娘出身名门，嫁给了某位公子。那人生性放荡，家产和嫁妆都被挥霍一空，逼迫六娘做妓女。六娘起先不从，涕泗横流，悲痛欲绝。老鸨用计玷污六娘的身子，于是六娘流落风尘。 　　后来六娘住在船上，卖掉首饰修补屋子，种竹浇花，过着恬淡的生活。她爱慕王初桐的才思，可惜不能托身于他，画了一幅人物小品（肖像）送给了他。图中用“天寒翠袖薄，日暮倚修竹”（杜甫《佳人》最后两句）补景，竹子署上“绝代佳人”四字，带入京城。当时著名文人都纷纷题诗称颂。但凡见过六娘（此处“崔徽”原指唐代歌妓，借用指六娘）的人都为之肝肠寸断（“萧娘一纸书”出处：《崔娘诗》唐·杨巨源 清润潘郎玉不如，中庭蕙草雪消初。风流才子多春思，肠断萧娘一纸书。） 　　王初桐也有给六娘写词。
