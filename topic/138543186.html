二、访谈节目分类及评述

上述提名是人民喜闻乐见的口碑作品，个别豆瓣评分不高但是群众基础较好。

此外，近年有些表现亮眼的节目，楼主认为很有发展潜力，也是很希望能拓展的方向！我最期待的访谈节目，分为以下三类：

1⃣️技能/知识输出类

这类节目以轻科普风格为主，需要业界标杆和口碑人物输出内容和经验。重干货，受众窄，毕竟观赏性不如娱乐向综艺，比如表演者言。不过这类节目太少了。

今日影评·表演者言 第一季 (2017)
8.8
2017 / 中国大陆 / 脱口秀 / 周迅 黄渤
一本好书 (2018)
9.1
2018 / 中国大陆 / 真人秀 / 关正文 / 赵立新 黄维德

🔥嘉宾：口碑戏骨，奖项/代表作/群众基础，三个得占俩。

❌短板：从节目制作方角度看，这类嘉宾往往对节目逼格要求高。他们形象维护比较严格，很多人甚至无高曝光和话题度需求，也不想说教，不一定豁得出去…

稀有指数：🌟🌟🌟🌟🌟

部分创作者去综艺节目输出了经验。众所周知的有：我就是演员、演员的诞生、演员的品格。

我就是演员 (2018)
6.7
2018 / 中国大陆 / 真人秀 / 吴彤 / 章子怡 徐峥

我不喜欢这类节目，他们展示的未必是对观众有用的。这类节目最后都变成粉丝自我感动和谩骂撕逼的工具。反正，离艺术审美提升差了远了…

仔细想想，这节目里，唯一艺术表现高光教科书现场，难道不是陈凯歌上台给徐娇讲戏吗？

2⃣️轻量级文化类

这类节目稍微轻松一些，输出的是软性意识和观点，比干货类容易欣赏和接受。

细分为脱口秀和访谈。

从数量上看，脱口秀多一些，表现形式为辩论、讨论和单口（误。

这俩个例里面，十三邀出发点很好，问答质量和信息密度比较高，气质稳重成熟；透明人比较年轻，访谈对象很有争议性，希望姜思达推下一季。

🔥嘉宾：个性鲜明的个人和群体代表。有智识会表达，有争议性和话题度。

❌短板：过审有点困难？再就是，嘉宾人选数量上有点缺乏？感觉上着上着就没了。

目前，这节目跟明星没啥关系，因为画风不合。客观上，有智识的娱乐圈公众人物是少数。谈吐不行的人不能硬上，容易暴露双商和逻辑硬伤。再退一步，明星也不需要艹有文化人设，站不住脚是小事，打脸引发群嘲形象大跌是大损失，得不偿失。此处，可援引组内人设崩塌现场神帖…

稀有度：🌟🌟🌟🌟

3⃣️硬核重量级文化类

这种还得看权威媒体了，没有钱和话语权，是拍不出来的。

不说访谈，放眼综艺和纪录片大类，像朗读者、中华诗词大会、舌尖上的中国这样的，不是谁想拍就拍，要拍就几乎都带宣传意味，各种传递正能量。

最近的要数国家宝藏了，不过有时没啥水平的人来蹦哒，非常尴尬。

国家宝藏 第一季 (2017)
9.0
2017 / 中国大陆 / 历史 真人秀 / 于蕾 / 张国立 李晨

要说重量级访谈，那就更难有了。话题内容有限制呗，反正是不能乱说话，说了就封。

锵锵三人行 (1998)
9.6
1998 / 中国香港 / 脱口秀 / 毕蜂 / 窦文涛 孟广美

锵锵三人行，经典中的经典，凤凰出精品。因为窦文涛和梁文道，我对楼里提到的听说、一千零一夜、局部都加了一层滤镜。

千禧之旅 (2000)
8.9
2000 / 中国香港 / 纪录片 / 余秋雨 曾瀞漪

千禧之旅，是楼里鹅提的，2000年凤凰的作品。看简介，这是文化学者余秋雨对中国文化遗址和景观的游历和评论，相当于千年一叹和文化苦旅的影视版本，太古早了，那时候我才3岁。

🔥嘉宾：文化大师，真才实学的专家教授。当然了高晓松这样的，勉强也可以接受。

❌短板：审查不容易过，一旦做成阉割版聊胜于无。

稀有度：🌟🌟🌟🌟🌟🌟🌟
